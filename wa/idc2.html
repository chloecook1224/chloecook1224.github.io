BMO Tamagotchi

About My Project

This project is a BMO Tamagtohci. You can feed BMO, pet BMO, and clean BMO all with the RFID chips. There is also an option to open up a menu screen to see how BMO is doing in terms of hunger, happiness and cleaniness. A personal little BMO!

Project Showcase

Parts Used:

-1 OLED
-1 Mini RFID Reader
-3 RFID Chips
-1 RFID Card
-1 Arduino Pro Mini

Moving Forward

This project has been a great learning experience. While it isn’t fully functional in its final form due to some issues with soldering and broken components, it works well on a breadboard. I'm proud of the coding skills I’ve developed and the variety of Arduino parts I've learned to use. In future iterations, I’m determined to improve the soldering, making it more functional and clean, while also learning more about soldering tips and tricks to create a more polished, fully functional product.
